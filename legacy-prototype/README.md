# MiniMarkdown Studio

一个接近 Typora 使用方式的 Markdown 编辑与阅读客户端原型。

## 已实现

- 多文档侧边栏与本地自动保存
- Markdown 实时编辑与阅读预览
- 标题大纲导航
- 本地 Markdown 文件打开与保存
- HTML 导出
- `Paper / Ink / Sepia` 三套阅读主题
- 双栏布局与专注模式切换

## 运行方式

直接在浏览器打开 `index.html` 即可使用。

如果你更希望通过本地服务启动，也可以在项目目录中运行任意静态文件服务。

## 说明

为了保证离线可运行，这个版本没有依赖外部 Markdown 解析库，而是内置了常用语法支持：

- 标题
- 段落
- 无序 / 有序列表
- 引用
- 分割线
- 粗体 / 斜体 / 删除线 / 行内代码
- 代码块
- 链接 / 图片
- 表格
